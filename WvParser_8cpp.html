Detailed Description

Defines WvParser class, which iterates over a corpus of sentences.

The WvParser class iterates over a corpus of sentences, encoding each word or punctuation symbol as a Word2Vec vector or "special" vector. It implements the functionality of the WvCorpusIterator class, separated out so it can be used by other classes.
